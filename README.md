## 🌟 Project Overview

> **A comprehensive Power BI dashboard** that transforms raw loan application data into actionable business intelligence — helping banks make smarter, data-driven lending decisions.

This project analyzes **563 real loan applications** across 6 interactive dashboard pages, uncovering patterns in demographics, credit risk, financial behavior, and property areas.

## 📈 Key Business Findings

```
✅ APPROVALS FAVOUR:
   ├── Good Credit History (Credit = 1.0)    → 79% approval
   ├── Semiurban Property Area               → 78.4% approval
   ├── Graduate Applicants                   → Higher avg loan term
   └── Non-Self-Employed                     → 70% vs 65% approval

❌ REJECTIONS LINKED TO:
   ├── Poor Credit History (Credit = 0.0)    → 93.5% rejection
   ├── Rural Property Area                   → Lowest approval (60%)
   ├── Low Income Group                      → Avg loan only ₹76K
   └── Self-Employment without co-applicant income
```
