# 📌 Project Overview

Financial institutions process thousands of loan applications daily.  
This project focuses on analyzing loan application data to uncover patterns, identify approval factors, and generate actionable business insights.

This is a complete **End-to-End Data Analytics Project** that includes:

✅ Data Cleaning  
✅ Data Preprocessing  
✅ Exploratory Data Analysis (EDA)  
✅ SQL Analysis  
✅ Data Visualization  
✅ Dashboard Development  
✅ Business Insights  

---

# 🎯 Business Problem

Banks need to understand:

- Which customers are more likely to get loan approval?
- What factors impact loan approval decisions?
- How can risk be minimized using data analytics?
- Which customer segments are safer for lending?

This project solves these questions using data-driven analysis.
